== Description ==

Enyoi is a versatile WordPress theme perfect for blogs, beauty salons and personal websites. It features a clean, modern design with responsive layout that looks great on all devices.

Key Features:
* Responsive design
* Custom menu support
* Featured images
* Threaded comments
* Translation ready
* Fast loading
